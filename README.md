# Proyectos y Tareas - Universidad

> Repositorio con los trabajos prácticos, proyectos y ejercicios desarrollados durante mi formación en Ingeniería de Sistemas.

---

## Descripción

Este repositorio contiene una colección de proyectos, tareas y ejercicios académicos realizados a lo largo de mi carrera universitaria. Incluye desarrollos en diferentes lenguajes y tecnologías, abarcando desde aplicaciones web sencillas hasta scripts de automatización.

El objetivo de este repositorio es documentar mi progreso académico y servir como portafolio de las habilidades adquiridas en la universidad.

---

## Tecnologías Utilizadas

| Tecnología | Descripción |
|------------|-------------|
| **Python** | Desarrollo de scripts, lógica de backend y automatización |
| **HTML5** | Estructura de páginas web |
| **CSS3** | Estilos y diseño visual |
| **JavaScript** | Interactividad y lógica en el frontend |
| **Flask** | Framework web para Python (posiblemente usado en `app.py`) |
| **Git** | Control de versiones |
